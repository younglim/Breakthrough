Web Services :: Breakthrough
Bring the Game to new places with RESTful JSON-output web services.
Both GET and POST parameters are accepted.


There are 5 web services available.

  1. Register
  2. Add Bot Code
  3. Get My Bot Codes
  4. Get All Bot Codes
  5. Play Game

1. Register

Service Endpoint: ⁄services⁄register.jsp

Parameters:

  • username
  • password

Example:

  • Request: ⁄services⁄register.jsp?username=mary&password=pw123456

  • Response: {"username":"mary","status":"Successfully registered user."}

2. Add Bot Code

Service Endpoint: ⁄services⁄addBotCode.jsp

Parameters:

  • botCode
  • username
  • password

Example:

  • Request: ⁄services⁄addBotCode.jsp?botCode=String playGame(...){return ...;}&username=mary&password=pw123456

  • Response: {"username":"mary","status":"Successfully created bot.","botCodeId":"5066549580791808"}

3. Get My Bot Codes

Returns user's created bot.

Service Endpoint: ⁄services⁄addBotCode.jsp

Parameters:

  • username
  • password

Example:

  • Request: ⁄services⁄getMyBotCodes.jsp?username=mary&password=pw123456

  • Details of Response JSON Elements:
    • "username": "mary"
    • "status": "Successfully retrieved bots."
    • "botCodes": [Array of bot codes with id, botCode, language, user, date]

5. Play Game

A game will be played between 2 bots, botCodeWhite and botCodeBlack. Specify the Long Botcode ID for each bot.
Wins, Loses and Draws will be recorded in the leaderboard.

Service Endpoint: ⁄services⁄playGame.jsp

Parameters:

  • botCodeWhite
  • botCodeBlack
  • username
  • password

Example:

  • Request: ⁄services⁄playGame.jsp?username=mary&password=pw123456&botCodeWhite=5629499534213120&botCodeBlack=5066549580791808

  • Details of Response JSON Elements:
    • "username": "mary"
    • "status": "Game Ended, White Wins, mary wins."
    • "moves": [Array of moves with each step taken (Step 0 to 100)]
